[`moptipyapps`](/moptipyapps)&nbsp;is a Python library that implements algorithms, benchmark instances, search spaces, encodings, and other tools for various important problems from the domain of optimization and operations research based on the [`moptipy`](/moptipy) package.
[`moptipyapps`](/moptipyapps)&nbsp;implements tools for the Traveling Salesperson Problem&nbsp;({%- include taglink.liquid tag="TSP" -%}), the Quadratic Assignment Problem&nbsp;({%- include taglink.liquid tag="QAP" -%}), two-dimensional bin packing, the Traveling Tournament Problem&nbsp;({%- include taglink.liquid tag="TTP" -%}), and production scheduling / material flow control tasks.
