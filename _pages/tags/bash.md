Bash is the shell used under [Ubuntu](https://ubuntu.com) {% include taglink.liquid tag="linux" %}, i.e., the program that "runs" in the terminal and interprets your commands.
It allows you to start and interact with other programs via the command line.
Learn more at <https://www.gnu.org/software/bash>.
A collection of all of my personal-use Bash scripts can be downloaded [here](http://thomasweise.github.io/scripts/scripts.tar.xz).
