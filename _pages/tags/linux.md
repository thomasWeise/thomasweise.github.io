[Linux](https://www.linux.org) is the leading open source operating system, i.e., a free alternative to Microsoft Windows or MacOS.
I recommend using it to all of my students, be it for studying, for software development, or for research.
Its variant [Ubuntu](https://ubuntu.com) is particularly easy to use and install.
While it offers a nice graphical user interface, you can also type commands directly in the terminal running the {% include taglink.liquid tag="bash" %} shell.
